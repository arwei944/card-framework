🏷️ 声明式 HTML 接入

纯 HTML 写 cf-card 标签，展示所有内置卡片类型，不需要写 JS 代码

← 返回示例列表

内置卡片类型

  • text - 文本卡
  • task - 任务卡
  • image - 图片卡
  • list - 列表卡
  • progress - 进度卡

使用方式

<cf-card 
  type="text" 
  title="标题">
  内容
</cf-card>

特点

  • ✅ 零 JavaScript 代码
  • ✅ 原生 HTML 标签语法
  • ✅ 自动初始化
  • ✅ 支持嵌套内容

这是一张文本卡片，支持丰富的 HTML 内容。


支持的标签：

  • 段落、标题
  • 列表、链接
  • 粗体、斜体
  • 图片、表格
  • 列表项 1
  • 列表项 2
  • 列表项 3
    •

    直接在 HTML 中使用 <cf-card> 标签即可。


    CardFrame 会自动检测页面中的自定义元素并初始化。
