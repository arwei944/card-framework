CSP 说明

CardFrame 兼容 Content Security Policy， 不使用 eval、new Function 或内联事件处理。

CardFrame 内置多层安全防护机制，保护应用免受 XSS 等攻击。


安全特性：

  • 🔒 HTML 净化 - 移除危险标签和属性
  • 🔗 URL 安全 - 检查协议，阻止 javascript:
  • 🎨 样式净化 - 移除危险的 CSS 表达式
  • 📜 CSP 兼容 - 支持严格的内容安全策略
