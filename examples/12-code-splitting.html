方式一：使用完整打包版（推荐入门使用）

<!-- 直接引入完整版本 --> <script src="dist/card-framework.js"></script> <link rel="stylesheet" href="dist/card-framework.css"> // 使用方式不变 const frame = new CardFrame('#container');

方式二：按需加载（生产环境优化）

<!-- 先加载核心模块和加载器 --> <script src="dist/core.js"></script> <script src="dist/loader.js"></script> <link rel="stylesheet" href="dist/card-framework.css"> // 按需加载需要的模块 async function initApp() { // 加载渲染模块（自动处理依赖） await CardFrame.load('render'); // 加载验证模块 await CardFrame.load('validation'); // 创建框架实例 const frame = new CardFrame('#container'); }

📊 模块依赖关系

  • core - 核心模块（必选）：Store, Utils, EventBus, TypeRegistry, FeedbackSystem
  • render - 渲染模块（依赖 core）：Renderer, LayoutEngine
  • validation - 验证模块（依赖 core）：RealTimeValidator, AutoFixer
  • security - 安全模块（依赖 core）：Security, CircuitBreaker
  • extras - 扩展模块（依赖 core, render）：ThemeManager, I18nManager, RelationshipEngine
  • plugins - 插件模块（依赖 core）：PluginManager
  • perf - 性能模块（依赖 core, render）：Perf, VirtualScroller
